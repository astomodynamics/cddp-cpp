# Example Data

`mpcc_racing_track.csv` is vendored for the Python MPCC portfolio demo.

Provenance:
- Derived from track data in [`alexliniger/MPCC`](https://github.com/alexliniger/MPCC)
- Method reference: Liniger, A., Domahidi, A., and Morari, M., "Optimization-based autonomous racing of 1:43 scale RC cars," Optimal Control Applications and Methods, 2015. doi: `10.1002/oca.2123`

The portfolio demo in this repository uses a simplified kinematic contouring-control setup for visualization and Python binding showcase purposes.
